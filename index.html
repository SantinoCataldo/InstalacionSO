Instalación SO - Tutorial

Por Santino Cataldo

Tutorial detallado y claro sobre cómo instalar Windows en una notebook, accesible para usuarios con conocimientos básicos y avanzados en tecnología.

1. Preparación

Requisitos de Hardware:

  • Procesador: 1 GHz o más rápido
  • RAM: 2 GB para 32 bits o 4 GB para 64 bits
  • Espacio en disco: 32 GB o más

Software y Materiales:

  • Imagen ISO de Windows: Descargar Windows 10
  • Unidad USB de 8 GB o más
  • Programa Rufus: Descargar Rufus

2. Creación de un USB Booteable

  1. Descarga e instala Rufus desde el enlace proporcionado.
  2. Inserta tu unidad USB en la notebook.
  3. Abre Rufus y configura:
    • Dispositivo: Selecciona tu unidad USB
    • Tipo de arranque: Selecciona la ISO de Windows
    • Esquema de partición: GPT
    • Sistema destino: UEFI (non CSM)
  4. Haz clic en "EMPEZAR" y espera a que termine el proceso

3. Configuración del BIOS/UEFI

  1. Reinicia tu notebook.
  2. Durante el arranque, presiona la tecla para acceder al BIOS/UEFI (comúnmente F2, Del, o Esc).
  3. Busca la sección "Boot" o "Arranque".
  4. Cambia el orden de arranque para que el USB sea la primera opción.
  5. Guarda los cambios y sal del BIOS/UEFI. Pulse F10 y, a continuación, pulse Y para Guardar cambios y Salir.

4. Instalación de Windows

  1. Arranca desde el USB.
  2. En la pantalla "Instalar Windows":
    • Selecciona idioma, formato de hora y teclado.
    • Haz clic en "Instalar ahora".
  3. Introduce la clave de producto si la tienes, o selecciona "No tengo clave de producto".
  4. Selecciona la versión de Windows que deseas instalar.
  5. Acepta los términos de licencia.
  6. Selecciona "Instalación personalizada".
  7. Elige el disco donde instalarás Windows.
  8. Espera a que se complete la instalación (la notebook se reiniciará varias veces).
  9. Configura las opciones iniciales de Windows (cuenta de usuario, privacidad, etc.).

5. Post-Instalación y Configuración Adicional

  1. Actualiza Windows:
    • Ve a Configuración > Actualización y seguridad > Windows Update.
    • Haz clic en "Buscar actualizaciones" e instala todas las disponibles.
  2. Instala los controladores:
    • Usa Windows Update para controladores básicos.
    • Visita el sitio web del fabricante de tu notebook para controladores específicos.
  3. Instala software esencial:
    • Navegador web (ej. Chrome, Brave)
    • Suite de oficina (ej. Microsoft Office)
    • Antivirus (Windows Defender viene preinstalado)
